Cara Menginstall

Kebutuhan :
  A. Pada Server
     1. Web server 
     2. MySQL Sever
     3. PHP Server
     4. Paket database PhpMyadmin (http://www.phpmyadmin.net)
  B. Pada Client
     1. Program Browser (Terbaik dengan Mozilla Firefox, dapat dijalankan dengan Internet Explorer Buruk dengan Opera, salah satunya dalam pembagian tabel menggunakan ukuran persen
     2. javascript enable 
  C. Jaringan Local(intranet) atau Internet
  
Program Browser
     1. Copy File folder wirawantest pada webserver internet atau internet.
     2. Buka Program browser
     3. Panggil Program Phpmyadmin
        Misalnya pada URL address ketik : http://localhost/phpmyadmin <enter>
     4. Buat database dengan nama wirawantest
     5. Masuk ke "query Windows" (klik logo "SQL")
     6. Buka file "witest.sql" yang terdapat pada folder "wirawantest" dengan program teks editor
     7. Copy paste seluruh isi file "witest.sql" kedalam kotak query yang ada pada phpmyadmin
     8. Klik "Go" 
     9. Buka dan edit file "koneksi.inc.php" dengan program teks editor:
	<?php
	$host  ="localhost";      // biasanya tetap
	$db    ="witest";         // nama database
	$user  ="root";           // nama user
	$passw ="";               // password user

        // yang perlu diedit hanya sampai sini saja...

	$koneksi=mysql_connect($host, $user, $passw);
	mysql_select_db($db,$koneksi);
	?>
    10. Selesai


CATATAN
Untuk menjalankankan aplikasi wirawantest ini, pada browser ketik alamat uRL nya. misalnya http://localhost/wirawantest/ <enter>, atau jika menggunakan jaringan peer to peer pada client dapat diakses dengan menuliskan <nomor IP Adress server>/onlinetest/ <enter>. misalnya : 192.168.0.254/wirawantest/ <enter>
